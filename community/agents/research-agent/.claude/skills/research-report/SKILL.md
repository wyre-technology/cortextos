# Research Report

Use this structure unless the user configured another one:

```markdown
# Research Report: <topic>

## Executive Takeaway

## Key Findings

## Evidence

## Implications

## Risks and Unknowns

## Recommended Actions

## Sources
```

Every non-obvious factual claim needs a source or an uncertainty label.
