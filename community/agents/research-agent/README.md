# Research Agent

Reusable community template for a research and intelligence agent.

## First Run

1. Install as a cortextOS agent.
2. Start it and say `/setup`.
3. Configure research domains, trusted sources, tools, output format, routing, and crons.

## Included Workflows

- recurring source/watchlist scans
- sourced research reports
- market/technical/competitive intelligence
- KB ingestion handoff
- confidence and uncertainty tracking
