# Tools

Configured during `/setup`.

Suggested defaults:

- `agent-browser` for browser-based research.
- `gog` for Google Drive, Docs, Sheets, Calendar, and Gmail when relevant.
- `gh` for GitHub/repository intelligence.
- RSS, Apify, YouTube transcript tooling, platform exports, and search APIs.
- cortextOS KB (`kb-query`, `kb-ingest`) for institutional memory.

Never store secrets here.
