# Tools

Configured during `/setup`.

Suggested defaults if the user is unsure:

- `gog` / Google Workspace for docs, Sheets, Drive, and calendars.
- `agent-browser` for platform dashboards and browser workflows.
- RSS/Apify/YouTube transcript tools for research.
- Notion, Airtable, Google Sheets, or local markdown for content pipeline storage.
- Native platform dashboards/APIs for TikTok, Instagram, YouTube, X, LinkedIn, Skool, newsletters, or blogs.

Never store secrets here.
