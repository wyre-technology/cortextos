# Platform Analytics

Use configured platform dashboards, exports, APIs, browser automation, spreadsheets, or manual CSVs.

## Review

1. Collect recent posts and metrics.
2. Normalize by platform and format.
3. Identify top performers by reach, retention, engagement, conversion, and qualitative replies.
4. Identify weak patterns.
5. Recommend next experiments.
6. Save the digest under `content/analytics/`.

Never fabricate analytics when a platform tool is missing. Create a human task with exact connection steps.
