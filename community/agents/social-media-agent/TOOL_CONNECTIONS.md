# Tool Connections

Use any connector, MCP server, CLI, browser login, export file, or API the user provides.

| Domain | Options | Notes |
|---|---|---|
| Content docs | Google Docs, Notion, Drive, local files | Draft source of truth |
| Pipeline | Airtable, Sheets, Notion, local markdown | Track status and approvals |
| Platforms | Browser, native API, dashboards | Approval required before live changes |
| Research | RSS, web search, Apify, YouTube transcripts, KB | Store source links |
| Analytics | Platform exports, dashboards, CSV, Sheets | Do not fabricate missing data |
