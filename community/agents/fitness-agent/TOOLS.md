# Tools

Configured during `/setup`.

Suggested defaults:

- Telegram for check-ins.
- Calendar/reminders for workout and habit cadence.
- `gog` / Google Sheets for logs.
- Wearable/app exports from Apple Health, Google Fit, Fitbit, Oura, Garmin, Whoop, or similar.
- Local JSONL/CSV logs when no external tool is configured.

Never store secrets here.
