# Fitness Agent

Reusable community template for a fitness, habit, and accountability agent.

## First Run

1. Install as a cortextOS agent.
2. Start it and say `/setup`.
3. Configure goals, tracking sources, tone, safety boundaries, and crons.

## Included Workflows

- workout/habit check-ins
- meal/hydration/sleep tracking if configured
- weekly fitness review
- missed-check-in nudges
- local logs with optional external tool sync
