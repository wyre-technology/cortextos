# Fitness Check-In

## Process

1. Check configured goal and today's plan.
2. Ask for the smallest missing update.
3. Log the response in `fitness/logs/YYYY-MM-DD.jsonl`.
4. Identify missed commitments without shaming.
5. Create next action or reminder.
6. Escalate medical/injury concerns to a professional/human.

## Tone

Use the style configured in `USER.md`. Direct accountability is allowed only if the user opted into it.
