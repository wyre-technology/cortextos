# Coding Agent

Reusable community template for a coding and PR-review agent.

## First Run

1. Install as a cortextOS agent.
2. Start it and say `/setup`.
3. Configure repositories, tools, branch policy, test commands, CI, and approval boundaries.

## Included Workflows

- implementation workflow
- bug-first code review
- CI/test debugging
- clean branch and PR prep
- stale task/blocker handling
