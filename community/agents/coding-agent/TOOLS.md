# Tools

Configured during `/setup`.

Suggested defaults:

- `git`, `gh`, and GitHub app/connector for repository workflows.
- `rg`, `jq`, project package managers, and language test runners.
- `agent-browser` or Playwright for browser/E2E validation.
- CI logs from GitHub Actions or the configured provider.
- cortextOS tasks, approvals, and save-output deliverables.

Never store secrets here.
