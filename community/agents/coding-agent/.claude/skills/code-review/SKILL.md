# Code Review

Findings first, ordered by severity.

Focus on:

- behavioral regressions
- missing tests
- broken contracts
- data loss/security/privacy risks
- concurrency/lifecycle issues
- inconsistent local patterns

If no issues are found, say so and name residual risk/test gaps.
